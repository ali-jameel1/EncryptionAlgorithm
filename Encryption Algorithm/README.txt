Author: Ali Jameel 


Purpose: Allow users to be able to encrypt and decrypt messages provided they enter a key and a message to either encrypt or decrypt.

Source and header files:

- encryption.c



Compilation and launching instructions

Step 1: gcc -o encryption encryption.c
Step 2: ./encryption
Step 3: Enter 1 for encrypting a message, enter 2 for decrypting a message
Step 4: Enter a partial key
Step 4: Enter either plaintext to be converted to decrypted text or enter cipher text to be converted into english.
Step 5: View the output of the program
